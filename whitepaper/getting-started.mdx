## Getting Started

### For Users

1. **Join a SUI Flow-enabled server**
2. **Use `/wallet` to open your wallet**
3. **Deposit SUI to your generated address**
4. **Start swapping, tipping, and trading!**

### For Server Admins

1. **Invite SUI Flow to your server**
2. **Contact the team for server activation**
3. **Use `/menu` to deploy the interactive wallet panel**
4. **Configure in any text channel for community access**

### Commands Overview

| Command             | Description                               |
| ------------------- | ----------------------------------------- |
| `/wallet`           | Open your personal wallet dashboard       |
| `/menu`             | (Admin) Deploy SUI Flow menu to a channel |
| `/pools`            | View and manage liquidity positions       |
| `/tip @user amount` | Send SUI tip to a community member        |
